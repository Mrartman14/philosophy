# Archive — Historical Implementation Plans

Historical, shipped one-shot implementation plans — kept for provenance.

Canonical/live design specs are in `docs/superpowers/specs/`; the active roadmap is `docs/superpowers/plans/2026-06-16-comprehensive-review-roadmap.md`.

## Status-header convention

Future plans should carry a status header near the top, e.g.:

```
Status: active | shipped | archived
```

This prevents accumulation of stale plans in the active `plans/` folder.
