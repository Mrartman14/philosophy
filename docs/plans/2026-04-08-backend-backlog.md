# Backend — Отложенные задачи

Задачи на будущее, без инструкций по реализации.

## Хранилище

- S3Storage реализация интерфейса Storage
- Presigned URL с настраиваемым TTL
- Миграция файлов с локального диска на S3

## База данных

- Миграция с SQLite на PostgreSQL

## API

- Пагинация списка лекций
- Фильтрация/поиск по лекциям
- Версионирование API (v1, v2)

## Инфраструктура

- CI/CD пайплайн
- Мониторинг и health-check эндпоинт
- Rate limiting
- Бекапы БД и файлов

## Фронтенд-интеграция

- OpenAPI спецификация + генерация TypeScript-типов для фронтенда
- Переключение фронтенда с локальных данных на API
